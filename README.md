# FinanceFlow Pro

This is your independent version of FinanceFlow, disconnected from AI Studio.

## Local Development

**Prerequisites:** Node.js (v18+)

1.  **Install dependencies:**
    `npm install`
2.  **Configure Environment:**
    Ensure you have a `.env.local` file with:
    *   `SUPABASE_URL`
    *   `SUPABASE_ANON_KEY`
    *   `API_KEY` (Gemini API Key)
3.  **Run the app:**
    `npm run dev`

## Deployment

Deploy this repository to Vercel for production use.
